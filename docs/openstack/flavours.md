Flavours
========

The following virtual machine flavours are supported on Ilifu:

| Name			| VCPUS	| RAM (GB)	| Disk (GB)   |
|----- 	 		|       |     	  	    | 	    |
| ilifu-A	       	| 1	| 4   	    |	20	    
| ilifu-B		| 2	| 8   	    |	20	    |
| ilifu-C		| 2	| 8   	    |	20	    |
| ilifu-D		| 4	| 32   	    |	20	    |
| ilifu-E		| 8	| 64   	    |	20	    |
| liifu-F		| 16	| 124       |	20	    |
| liifu-G		| 32	| 250       |	20	    |
| liifu-H		| 32	| 500       |	20	    |
